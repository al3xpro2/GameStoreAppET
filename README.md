## Funcionalidades
*   Gestión de productos (CRUD).
*   Autenticación de usuarios (login/signup).
*   Gestión de carrito de compras.
*   Creación y visualización de órdenes.
*   Actualización de estado de órdenes para administradores.

## Endpoints
### Endpoints propios
La URL base para la API es: https://x8ki-letl-twmt.n7.xano.io/api:NEoI9qGr/

*   GET /product: Obtiene la lista de productos.
*   POST /product: Crea un nuevo producto.
*   PATCH /product/{id}: Actualiza un producto existente.
*   DELETE /product/{id}: Elimina un producto.
*   POST /auth/login: Inicia sesión.
*   POST /auth/signup: Registra un nuevo usuario.
*   GET /carrito: Obtiene el contenido del carrito del usuario.
*   POST /carrito: Actualiza el carrito del usuario.
*   POST /orders: Crea una nueva orden.
*   GET /orders: Obtiene el historial de órdenes del usuario.
*   PATCH /orders/{id}: Actualiza el estado de una orden (admin).
